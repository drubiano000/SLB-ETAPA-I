En el archivo back_generación.py, al inicio, se pueden ajustar las variables:

path: para establecer la ruta de dónde se leen los archivos que alimentan el proceso y los archivos que resulten con las series de tiempo.

archivo_sd_fallas: "DOWNTIMES_ANDES.csv" con el nombre del archivo de datos SD y Fallas.

archivo_canales: "ESP_RAW_DATA_ANDES.csv" con el nombre del archivo de datos de los de los canales.
